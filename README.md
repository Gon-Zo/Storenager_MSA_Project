## MSA (Mobile Service API)
> Mobile API(Application Programming Interface,응용 프로그래밍 인터페이스는) <br/>
> 응용 프로그램(앱) 에서 사용할 수 있도록, 운영체제나 프로그래밍 언어가 제공하는 기능을 제어할 수 있게 만든 인터페이스를 뜻합니다.


## Use To 

- Spring
- Java 
- Mybatis 
- Oracle
